Add hotel watch

Alerts fire when any matching hotel's price drops 20%+ below the lowest price ever recorded for it — not on the first sighting, which just sets the baseline. Prices are always EUR, per night. Review scores use Google's 5-point scale (Booking's "8+" ≈ 4.0+ here). Hostels/shared-room properties are excluded automatically by name matching — a best-effort heuristic, since Google's data doesn't expose a clean property-type field.
